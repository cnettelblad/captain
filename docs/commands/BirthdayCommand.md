# 🎂 Birthdays

Let the server celebrate with you — set your birthday (month and day) and get a shoutout when the big day rolls around.

## Usage

### Set your birthday
```
/birthday add [month] [day]
```
**Example:** `/birthday add march 14`

The month can be a number (**1–12**) or a name (`mar`, `march`, `March`).

### Remove your birthday
```
/birthday remove
```

## Notes
- Only the **month and day** are stored — your year stays private.
- Want to change it? Just run `/birthday add` again.
