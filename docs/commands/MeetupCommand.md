# 👥 Meetup

Commands for viewing your in-person meetup history. To actually **log** a meetup, use `/met`.

## Usage

### List everyone you've met
```
/meetup list
```
An ephemeral embed of your confirmed meetups. Click **Share** in the result to publish it to the channel.

## Related
- `/met user:@SomeUser` — log a new meetup and earn milestone roles
