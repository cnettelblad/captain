# 🌍 Countries

Track every country you've visited, compare trips with friends, and see who's the most well-travelled on the server.

## Adding & removing

### Add a country
```
/countries add [country] [date] [note]
```
**Examples:**
- `/countries add Japan`
- `/countries add 🇮🇹 2024-06-15 Rome & Florence`

You can enter a **name**, **ISO code**, or the **flag emoji**. `date` and `note` are optional.

### Remove a country
```
/countries remove [country]
```
**Example:** `/countries remove Japan`

## Exploring

### Your list
```
/countries list
```
Every country you've visited, with UN & ISO 3166 totals in the footer.

### Who's been there?
```
/countries who [country]
```
**Example:** `/countries who Japan`

Lists every server member who's set foot in that country.

### Compare with a friend
```
/countries compare [user]
```
**Example:** `/countries compare @Friend`

Shows what you have in common and what's unique to each of you.

### Most visited
```
/countries most-visited
```
The top 10 countries by server visits.
