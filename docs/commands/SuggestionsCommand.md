# 💡 Suggestions

Got an idea that could make **Wanderlust** better? Submit it here and staff will take a look.

## Submitting a suggestion

Click the **Submit a Suggestion** button on the suggestions prompt. Describe your idea (and optionally attach up to **5** images or files) — it'll be delivered straight to the staff log channel.

## Admin commands

### Post the suggestions prompt
```
/suggestions set [channel]
```
**Example:** `/suggestions set #your-channel`

Creates the "Submit a Suggestion" button in the chosen channel. **Requires the Manage Server permission.**

### Browse submissions
```
/suggestions list [page(default 1)]
```
**Example:** `/suggestions list 2`

Paginated list of every submitted suggestion, with timestamps and authors.
