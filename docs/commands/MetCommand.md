# 🤝 Have you met another Wanderluster?

Log your in-person meetups with other members and earn milestone roles based on how many fellow Wanderlusters you've met in real life.

## How it works

Use `/met` with the person you've met:
```
/met [user]
```
**Example:** `/met @SomeUser`

They'll receive a **DM** asking them to confirm or decline. If their DMs are closed, they can simply run the same command back at you instead (**Example:** `/met @You`).

## 🏅 Milestone Roles

You'll automatically earn a role at each of these thresholds:

- **1** meetup → <@&1230237976605360168>
- **5** meetups → <@&1230238066510139434>
- **10** meetups → <@&1230238118729351189>
- **25** meetups → <@&1352005935798947872>
- **50** meetups → <@&1380168192999030814>
- **100** meetups → <@&1380169001652719756>

## Rules
- You can't meet **yourself** or a **bot** (sorry).
- You can have up to **10 pending** meetup requests at once.
- Rejected requests can be re-sent after **14 days**.
