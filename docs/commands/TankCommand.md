# 🐟 Tank

Send a user to the tank for a set duration — or until they are freed.

## Usage
```
/tank [user] [duration] [reason]
```
**Examples:**
- `/tank @SomeUser`
- `/tank @SomeUser 1h`
- `/tank @SomeUser 1h Being too based`

## Duration format
- `30s` — seconds
- `5m` — minutes
- `1h` — hours
- `1d` — days
- **Omit** the duration to tank **indefinitely**.

## Requirements
- You need the **Moderate Members** permission.
- Bots can't be tanked.

## Related
- `/free user:@SomeUser` — release a tanked user early
