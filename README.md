# Captain Discord Bot

Discord bot for the Captain server.

## Last.fm Commands

Link your Last.fm account to view music statistics:

- `/fm set <username>` - Link your Last.fm account
- `/fm unset` - Unlink your account
- `/fm np [user]` - Show now playing or recent track
- `/fm recent [user] [limit]` - Show recent tracks (1-15)
- `/fm top <type> [period] [user]` - Show top artists, albums, or tracks
- `/fm profile [user]` - Show profile stats and scrobble count
- `/fm compare <user>` - Compare music taste with another user
